

# PyDucky Script

In order to use this tool you must have
your pico mounted in ```/media/YOURUSER/PICONAME``` 
for linux and in windows you must know the disk letter.

# To run the script:

Linux: 

```$ python3 pyducky_linux.py```

Windows: 

```$ python3 pyducky_windows.py```
